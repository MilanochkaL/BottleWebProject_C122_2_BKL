# Обучающее Web-приложение по теории графов

## Обзор проекта

Данный проект - веб-сайт, разработанный с использованием Python с фреймворком Bottle, предлагающий информацию по теории графов и алгоритмам. Веб-сайт будет иметь разделы, посвященные объяснению концепций нахождения вершины с наименьшим окружением, поиску подграфа в графе, а также нахождению эйлерова цикла или цепи в эйлеровом графе. Каждая из этих тем будет иметь калькулятор для пользователей, чтобы практиковать и проверять свои знания.

## Описание задания

Мы разрабатываем веб-приложение для работы с графами на языке программирования Python с использованием фреймворка Bottle. Задача включает создание страниц:

* **Авторы**: Раздел, предоставляющий информацию о авторах веб-сайта, включая их имена, учреждения и контактные данные.
* **Теория графов**: В-depth объяснение следующих концепций теории графов:
 + Нахождение вершины с наименьшим окружением
 + Поиск подграфа в графе
 + Нахождение эйлерова цикла или цепи в эйлеровом графе

А также следующего функционала:

* **Калькуляторы**: Интерактивные калькуляторы для каждой из упомянутых концепций, позволяющие пользователям вводить значения и видеть результаты.
* **Unit-тесты**: Автоматические тесты, чтобы обеспечить точность и надежность калькуляторов и алгоритмов теории графов.

## Требования к техническому обеспечению 
*  Установленная среда разработки Microsoft Visual Studio Community 2019 версии 16.8.4 или выше.
*  Установленный язык программирования Python (рекомендуется версия от 3.x).
*  Установленный фреймворк Bottle. 
*  Доступ к интернету для загрузки необходимых зависимостей и документации.

## Установка
1. Установите Python, если он не установлен.
2. Установите фреймворк Bottle с помощью pip install bottle.
3. Клонируйте репозиторий с проектом: git clone <URL репозитория>.
4. Перейдите в директорию проекта: cd BottleWebProject_C122_2_BKL.
5. Запустите веб-приложение: python app.py.

## Тестирование

1. Запустить unit-тесты: `python -m unittest discover tests`
2. Тестировать калькуляторы, вводя значения и проверяя результаты
